# Blog CMS

Each journal post is a Markdown file in this folder.

To publish a new journal, create a file like `my-new-journal.md`:

```md
---
title: "My Journal Title"
date: "2026-04-20"
slug: "my-journal-title"
coverImage: "/assets/source/p22_01_13ac514cfa_461x308.png"
preview: "A short preview sentence."
published: true
---

Write the journal text here.

Each blank line becomes a new paragraph.
```

Set `published: false` to keep a draft hidden.
